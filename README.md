# TaskFlow Full-Stack

## Structure du projet
- taskflow-backend/ : API REST Node.js/Express + MongoDB
- taskflow-react/ : Interface React (Front-End)

## Prérequis
- Node.js installé
- MongoDB installé localement (service Windows)

## Lancer le projet

### 1. Base de données
MongoDB démarre automatiquement comme service Windows.

### 2. Serveur Back-End
```bash
cd taskflow-backend
npm install
npm run dev
3. Interface Front-End
cd taskflow-react
npm install
npm run dev
Accès
 • API Back-End : http://localhost:5000
 • Front-End : http://localhost:5173
